FAQ

Questions, answered

Do I need a personal access token?

No. Authentication uses VS Code's built-in GitHub, GitHub Enterprise, and Microsoft (Azure DevOps) OAuth providers. The extension never asks for or stores tokens itself.

How do I connect GitHub Enterprise?

Set VS Code's github-enterprise.uri setting to your GHE host, then add a repo using its full URL, e.g. https://github.mycompany.com/team/service.

How do I add an Azure DevOps repository?

Paste the full repo URL, e.g. https://dev.azure.com/org/project/_git/repo. Sign-in uses your Microsoft account.

Where is my data stored?

Your watched-repo list is kept in VS Code's extension storage. OAuth tokens are managed by VS Code in the OS secret store.
